
<h1>Project Description</h1>

BLOGAPP is a blog application built using Next.js, React.js, and Node.js. It includes an API, admin panel, and multiple components to manage blog posts efficiently. The project follows a structured approach for maintainability and scalability.

<h1>Installation</h1>

Clone the repository:https://github.com/ashutosh-2403/BLOGAPP


Navigate to the project directory:
cd BLOG-APP

<h1>Install dependencies:</h1>

npm install

Set up the environment variables in a .env file.

Run the development server:

npm run dev

<h1>Usage</h1>

Creating Blogs: Users can create new blog posts from the admin panel.

Managing Blogs: Admin users can edit or delete blog posts through the dashboard.

Reading Blogs: Visitors can browse and read published blogs on the homepage.

Email Notifications: The system sends emails for important updates.

Responsive UI: The app is fully responsive, making it accessible on any device.
